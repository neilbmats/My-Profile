I am planning to make this project with an idea of receiving push notifications on my device every time motion is censored in my drawer but in case, I am not able to stick to the idea of push notifications, then I am planning to set it up with an alarm device that sets off as motion is detected. The cost of making this project is another reason why I decided to do this project as it is quite cheap. My plan is to complete this project under $50 but I am willing to make modifications to it along the way. I believe this project would be a good start to get an idea of security and the things that revolve around it. This project could also set me up with skills such as time management and security monitoring which could help me with the job I am aiming for. The main requirement of this project is being able to switch the security device on and off to prevent the device from setting of when not in use. The main reason why I choose this project is the feasibility. This project can be created without much difficulty with access to the right tools, technology and skill. Coding the Adrino module to respond to the motion sensor is something that I need to learn in order to complete this project.
